# Set-proxy-in-Linux

将下面的命令添加到`~/.bashrc`中，然后`source ~/.bashrc`

```bash
proxy () {
  export ALL_PROXY="socks5://192.168.111.1:7890"
  export all_proxy="socks5://192.168.111.1:7890"
  curl https://ip.gs
}

noproxy () {
  unset ALL_PROXY
  unset all_proxy
  curl https://ip.gs
}
```
